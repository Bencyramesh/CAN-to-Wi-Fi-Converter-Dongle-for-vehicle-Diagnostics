# CAN-to-Wi-Fi-Converter-Dongle-for-vehicle-Diagnostic

## Overview

This project is a custom designed CAN to Wi-Fi dongle intended for automotive applications. The device is designed to receive CAN bus messages from a vehicle or CAN network and transmit the data wirelessly over Wi-Fi using an ESP32.

The project focuses on learning and implementing automotive communication, embedded systems, and PCB design.

## Objectives

- Design a compact automotive communication PCB.
- Interface with CAN bus networks.
- Receive CAN messages from a vehicle or development board.
- Transmit CAN data wirelessly using Wi-Fi.
- Learn automotive PCB design and embedded firmware development.

## Features

- ESP32 based CAN to Wi-Fi communication
- CAN bus interface
- Custom PCB design using KiCad
- USB interface for programming and debugging
- Compact hardware design
- Future support for vehicle diagnostics and CAN data logging

## Hardware

- ESP32
- MCP2515 CAN Controller
- CAN Transceiver
- USB Connector
- Voltage Regulator
- Passive Components
